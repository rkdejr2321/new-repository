## Mockito란?
모의 객체 생성, 검증, 스텁을 지원하는 자바 모의객체 프레임워크  
mockito-core 모듈을 의존에 추가하면 사용할 수 있다.
~~~
dependencies {
    testImplemention('org.mockito:mockito-core:2.26.0)
}
~~~
